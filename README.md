# COOK
A highly customizable custom-wordlist generator.

# Installation
```
go get github.com/giteshnxtlvl/cook
```

# Usage
## Basic Usage
```
Structure:
cook -ingredient1 <value1> -ingredient2 <value2> ... [pattern]
```
